# RevatureCollections
How to install 
1. Maven install
2. add the following code to your pom.xml
````xml
    <dependency>
        <groupId>com.revature.collections</groupId>
        <artifactId>RevatureCollections</artifactId>
        <version>1.1</version>
    </dependency>
````
3. ```import  revatureCollections.*;```
4. profit
